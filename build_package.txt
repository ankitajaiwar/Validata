 Test Publish to TestPyPI
Before publishing to production PyPI, test it on TestPyPI:


python -m twine upload --repository testpypi dist/*


Login with your TestPyPI credentials.

To install from TestPyPI:


pip install --index-url https://test.pypi.org/simple/ validata-checker


5️⃣ Publish to PyPI
Once tested, publish to the real PyPI:


python -m twine upload dist/*
Login with your PyPI credentials.


6️⃣ Verify Installation
Test in a new environment:


pip install validata-checker
Then:


from validata.checker import DataQualityChecker
